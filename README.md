\# PetConnect

Proyecto desarrollado como evidencia individual del programa ADSO - SENA.

\## Descripción

PetConnect es una aplicación desarrollada en Java que permite registrar información básica de mascotas utilizando una base de datos MySQL mediante JDBC.

\## Tecnologías utilizadas

\- Java

\- NetBeans

\- Maven

\- MySQL

\- JDBC

\- Git

\- GitHub

\## Estructura del proyecto

\- Conexion.java

\- Usuario.java

\- Mascota.java

\- MascotaDAO.java

\- PetConnect.java

\## Funcionalidades

✔ Conexión a MySQL

✔ Registro de mascotas

✔ Programación Orientada a Objetos

✔ Encapsulamiento

✔ Uso de DAO

✔ Inserción de datos mediante JDBC
